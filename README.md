# item-index
Monumenta Item Index bot for Discord

#### Features
* ##### Items
    * Adding items
    * Removing items
    * Tagging system for items - add, remove, and modify tags
    * Looking up items
    * Referencing Mirror's Region 1 item spreadsheet

* ##### Kaul
    * Add and remove Kaul role
    * Ping system for Kaul fights
